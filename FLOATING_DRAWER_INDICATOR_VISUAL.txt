IMPLEMENTATION METHOD:

Each screen wraps main content in:
┌──────────────────────────────────────┐
│ Box(modifier.fillMaxSize()) {         │
│   ├─ LazyColumn/Column (main content)│
│   └─ DrawerSwipeIndicator (floating) │
│ }                                     │
└──────────────────────────────────────┘

This allows the indicator to float above
all content without affecting layout.
